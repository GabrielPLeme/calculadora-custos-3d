📌 Descrição do projeto

Sistema desenvolvido em Java com JavaFX para simular uma calculadora de custos de impressão 3D. O sistema permite calcular e exibir os principais custos envolvidos no processo de impressão, como materiais, energia, mão de obra, manutenção e falhas, apresentando um resumo organizado em interface gráfica.

O projeto tem como objetivo aplicar conceitos de Programação Orientada a Objetos (POO), manipulação de interfaces gráficas com JavaFX e organização de lógica de cálculo de custos, simulando um cenário real de produção.

🧠 Tecnologias utilizadas
Java
JavaFX
Programação Orientada a Objetos (POO)
Estrutura de interface com layouts (StackPane, HBox, etc.)
🎯 Funcionalidades
Cálculo de custo de materiais de impressão 3D
Estimativa de falhas (10%)
Cálculo de energia e máquina
Inclusão de mão de obra e manutenção
Interface gráfica com resumo organizado dos custos
